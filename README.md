# Student-Score-Predictor-Multivariable-Linear-Regression-Model

#Student Score Prediction Using Linear Regression

A machine learning project built to predict a student's final marks (in percentage) based on their age and daily study habits. The model utilizes a two-variable linear regression approach, developed following the completion of the first course in the Machine Learning Specialization.

## 📌 Project Overview
This project was developed during the 3rd semester (2nd year) of a Bachelor of Computer Application (BCA) in Artificial Intelligence and Data Analytics. 

The goal is to analyze how age and daily study commitment influence academic outcomes across various student demographics. The dataset spans a wide array of academic levels, capturing diverse learning patterns to train a robust predictive model.

## 📊 Dataset & Data Collection
The data powering this model was crowd-sourced and collected from various fields using **Google Forms** and organized via **Google Sheets**. The dataset includes data points from:
* Class 11 and 12 high school students
* Undergraduate college-going students
* Competitive exam aspirants (including high-tier exam candidates, such as those preparing for the Indian Space Research Organisation (ISRO))

### Model Variables:
* **Input Feature 1 ($X_1$):** Student Age
* **Input Feature 2 ($X_2$):** Student Average Hours of Study per day
* **Output Target ($Y$):** Student Final Marks (expressed as a percentage)
